# 📌 About The Project

Digital Kisan Hub is a **data-driven agriculture dashboard** built to simplify access to mandi price data and help users visualize agricultural market trends.

The platform provides **location-based crop price insights**, interactive charts, and a clean dashboard interface designed for exploring agricultural datasets efficiently.

---

# ✨ Key Features

🌾 **Mandi Price Tracking**
Access mandi price data for agricultural products across districts.

📊 **Price Trend Visualization**
Interactive graphs to analyze price trends over time.

📍 **District-Based Data Filtering**
Filter mandi prices by district across Maharashtra.

👤 **User Authentication**
Secure login system using Firebase Authentication.

📂 **Structured Agricultural Dataset**
Efficient data lookup using a local JSON dataset.

💻 **Desktop Optimized UI**
Currently optimized for **desktop screens**.

# 📊 Project Motivation

Agriculture generates massive amounts of market data, but much of it is **difficult for farmers to access or analyze**.

Digital Kisan Hub aims to bridge this gap by:

* Aggregating mandi price data
* Presenting it through **clean visual dashboards**
* Enabling **data-driven farming decisions**

# ⚙️ Run Locally

Clone the repository

```
git clone https://github.com/Vallabh-01/Project_Kisan_Hub.git
```

Navigate to project

```
cd Project_Kisan_Hub
```

Install dependencies

```
npm install
```

Start development server

```
npm run dev
```
